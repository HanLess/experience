<h3>普通登录流程</h3>

（1）通过手机号获取验证码，生成 token

（2）token入表，存缓存，缓存事件30天，且token解密后会得到有效时间

（3）token信息返回给前端，每次调取接口通过拦截器校验登录信息

（4）登录信息的校验是校验token是否有效，

<h3>绑定微信登录流程</h3>

（1）客户端需要拿到必要的信息，包括 openId，unionId，appId，code等以及用户手机号（可以加一步短信验证）

（2）用手机号查user

（3）将微信信息与user进行绑定

（4）在查询绑定关系的时候根据 用户信息 和 微信信息
