java中 synchronized 与 ReentrantLock 都是可重入锁。

可重入锁的作用：防止死锁。

场景：

（1）递归调用

（2）继承父类，重写方法时

```
public class Widget{
    public synchronized void doSomething(){
        ........
    }
}

public class LoggingWidget extends Widget{
    public synchronized void doSomething(){
        super.doSomething();
    }
}
```
