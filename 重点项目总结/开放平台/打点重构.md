通过装饰器封装打点逻辑

```
@sendLog(name)
componentDidMount () {}
```

```
@sendLog(name)
clickEvent() {}
```

```
sendLog (target, key, des) {
   let fn = des.value
   des.value = function (args) {
      
      log()
      
      fn.apply(this, ...args)
   }
}
```

装饰器是 aop 设计模式的实现，可以在打点、监听报错、判断登录、事件节流等基础功能中使用
