#### 避免读取超出数组长度的内容

读取超出数组长度的数据可能会对性能产生惊人的影响，例如，array.length === 5时读取array[42]。在这种情况下，数组索引42超出范围，数组本身不存在该属性，因此JavaScript引擎必须执行昂贵的原型链查找。一旦遇到这种情况，V8就会记住“需要处理特殊情况”，并且它再也不会像读取越界之前那样快。

#### 优先使用数组而不是类似数组的对象

类似数组的对象如下：

```
const arrayLike = {};
arrayLike[0] = 'a';
arrayLike[1] = 'b';
arrayLike[2] = 'c';
arrayLike.length = 3;
```
